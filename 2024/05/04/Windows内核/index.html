D/B位

对DS,ES等数据段：
D=1，段上限为4GB
D=0，段上限为64KB

对SS段：
D=1，使用32位堆栈指针寄存器
D=0，使用16位堆栈指针寄存器

对CS段：
D=1，采用32位寻址方式
D=0，采用16位寻址方式

G位

G=0 limit单位是字节
G=1 limit单位为4KB

汇编

CS：代码段寄存器 描述的代码
SS：栈段 描述栈，比如局部变量
DS：数据段 堆地址，全局变量 es一样的内容
ES 扩展段 串copy movsb so edi esi
FS 上下文环境段 R3代表我们TEB R0代码是KPCR
除了CS外，其他的段寄存器都可以通过mov，les，lss，lds，lfs，lgs指令进行修改
CS是代码段 CS的改变意味着EIP的改变，改变CS的同时必须修改EIP，所以无法使用上面的指令进行修改

Windbg常用命令

r :查看寄存器
eg:r gdtr 查看gdt表的指向
d :查看内存 db(查看1字节) 或者 dw(查看2字节) 或者 dd(查看4字节) 或者 dq(查看8字节) 对应查看字节byte、word、dword、qword
eg:dq 地址 L1 //L是limit 1是看1个
eg:dds 地址 //竖着看内存
e:写内存 eb(查看1字节) 或者 ew(查看2字节) 或者 ed(查看4字节) 或者 eq(查看8字节) 对应查看字节byte、word、dword、qword
eg:eq 地址(80b99048) 写入的字节(00cff300`0000ffff)
